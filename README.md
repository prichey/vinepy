vinepy
======

*Python wrapper for [Vine](https://vine.co) API*

## Requirements

* [requests](http://docs.python-requests.org/en/latest/)

## Installation

`pip install vinepy`

## Usage

```python
import vinepy

vine = vinepy.API(username='email@host.com', password='leinternetz')
user = vine.user
followers = user.followers()
timeline = user.timeline()
```
